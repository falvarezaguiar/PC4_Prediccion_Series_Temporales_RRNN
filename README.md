# PC4_Forcasting_LSTM_Rossmann
Ficheros Notebook para hacer EDA y pronósticos de series temporales.

Crearemos un notebook para realizar un EDA y visualización de los datos de Ventas de una tienda de Rossmann

El fichero de datos puede cambia por cada tienda, solo tendremos que cambiar el Path_Name en la primera celda.

Los ficheros que tendremos en Jupyter:

PC4_EDA_Visualizacion_Tienda6.ipynb  (undestanding the dataset, STL for the series data, review type & sort instanse)
PC4_Feature_Eng&Baselines.ipynb (split data in train & test, review the main variables, create some Features variables of memory and checking category variables (as promo, holidays)
